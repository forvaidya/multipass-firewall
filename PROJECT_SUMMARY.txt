🎯 WHAT YOU HAVE

A production-ready outbound firewall system with 3 deployment options:

  1. Simple       → nftables IP whitelisting only
  2. DNS-Based    → CoreDNS + nftables
  3. Three-Layer  → CoreDNS + eBPF + nftables ⭐ RECOMMENDED


📊 THREE-LAYER ARCHITECTURE (Recommended)

  Layer 1: CoreDNS (Domain Whitelisting)
    • Only whitelisted domains resolve
    • Blocked domains return NXDOMAIN
    • Block: pornhub.com, facebook.com, etc.
    • Allow: github.com, ubuntu.com, etc.

  Layer 2: eBPF/Falco (DNS Resolver Monitoring)
    • Monitor which DNS servers are used
    • Allow: 8.8.8.8, 1.1.1.1
    • Alert on unauthorized resolvers
    • Real-time detection

  Layer 3: nftables (IP Whitelisting)
    • Default deny all IPs
    • Only whitelisted IPs can be accessed
    • Kernel-level filtering (<1ms)
    • Backup defense if Layers 1-2 fail


🚀 QUICK START

  Option 1: Simple Setup
    sudo ./scripts/setup.sh --domains "github.com" --auto

  Option 2: With DNS Filtering
    sudo ./scripts/setup-coredns.sh --whitelist "github.com"

  Option 3: Three-Layer (Recommended)
    sudo ./scripts/setup-three-layer.sh \
      --whitelist "github.com,ubuntu.com" \
      --ips "140.82.113.4,185.125.190.81"

🔒 SECURITY MODEL

Default: DENY ALL (except whitelisted)

Whitelisted:
  ✓ Domains: github.com, ubuntu.com, registry.npmjs.org, pypi.org
  ✓ IPs: DNS resolvers (8.8.8.8, 1.1.1.1)
  ✓ IPs: Service endpoints (github, ubuntu, etc.)

  ✗ Social media (facebook.com, twitter.com, instagram.com)
  ✗ Unauthorized IP addresses

🛠️ CONFIGURATION

Edit Whitelist:
  Option 1: sudo vim /etc/falco-firewall/policy.yaml
  Option 2: sudo vim /etc/coredns/Corefile
  Option 3: sudo vim /etc/coredns/Corefile + /etc/nftables.conf

Reload Configuration:
  Option 1: sudo systemctl restart falco-firewall-enforce
  Option 2: sudo systemctl restart coredns
  Option 3: sudo systemctl restart coredns && sudo nft -f /etc/nftables.conf


🔍 MONITORING

Check Status:
  sudo systemctl status coredns
  sudo systemctl status falco
  sudo nft list chain inet filter firewall_out

View Logs:
  sudo journalctl -u coredns -f
  sudo tail -f /var/log/falco/falco.log
